Examples

Attached to a field with the format specified via options.

<input type="text" class="span2" value="11-30-2012" id="dp1">

<script type="text/javascript">
$(function() {
	$('#dp1').datepicker({ format: 'mm-dd-yyyy' });
});
</script>

Attachet to a field with the format specified via data tag.

<input type="text" class="span2" value="11/30/12" data-date-format="mm/dd/yy" id="dp2">

<script type="text/javascript">
$(function() {
	$('#dp2').datepicker();
});
</script>

<script type="text/javascript">
$(function() {
	var startDate = new Date(2012,11,26);
	var endDate = new Date(2012,11,30);
	$('#dp4').datepicker()
		.on('changeDate', function(e) {
			if (e.date.valueOf() > endDate.valueOf()) {
				$('#alert').show().find('strong').text('The start date can not be greater then the end date');
			} else {
				$('#alert').hide();
				startDate = new Date(e.date);
				$('#startDate').text($('#dp4').data('date'));
			}
			$('#dp4').datepicker('hide');
		});
	$('#dp5').datepicker()
		.on('changeDate', function(e) {
			if (e.date.valueOf() < startDate.valueOf()) {
				$('#alert').show().find('strong').text('The end date can not be less then the start date');
			} else {
				$('#alert').hide();
				endDate = new Date(e.date);
				$('#endDate').text($('#dp5').data('date'));
			}
			$('#dp5').datepicker('hide');
		});
});
</script>

Using bootstrap-datepicker.js

Call the datepicker via javascript:

$('.datepicker').datepicker()

Options

Name Type Default Description
format string 'mm/dd/yyyy' the date format, combination of d, dd, m, mm, yy, yyyy
weekStart integer 0 day of the week start. 0 for Sunday - 6 for Saturday
viewMode string | integer 0 = 'days' set the start view mode. Accepts: 'days', 'months', 'years', 0 for days, 1 for months and 2 for years
minViewMode string | integer 0 = 'days' set a limit for view mode. Accepts: 'days', 'months', 'years', 0 for days, 1 for months and 2 for years
language string 'en' set the language locale. Accepts: en — English, ab — Abkhaz, af — Afrikaans, sq — Albanian, am — Amharic, ar — Arabic, hy — Armenian, az — Azeri, be — Belarusian, bn — Bengali, bs — Bosnian, bg — Bulgarian, my — Burmese, ca — Catalan, ce — Chechen, chr — Cherokee, zh — Chinese, cr — Cree, hrv — Croatian, cs — Czech, da — Danish, nl — Dutch, eo — Esperanto, et — Estonian, fo — Faroese, fi — Finnish, fr — French, gl — Galician, ka — Georgian, de — German, el — Greek, ht — Haitian Creole, haw — Hawaiian, he — Hebrew, hi — Hindi, hu — Hungarian, is — Icelandic, io — Ido, id — Indonesian, ia — Interlingua, ga — Irish, it — Italian, ja — Japanese, roa — Jèrriais, krl — Karelian, kk — Kazakh, km — Khmer, ko — Korean, ky — Kyrgyz, lo — Lao, la — Latin, lv — Latvian, lez — Lezgi, lt — Lithuanian, liv — Livonian, jbo — Lojban, lb — Luxembourgish, mk — Macedonian, ms — Malay, ml — Malayalam, mt — Maltese, mn — Mongolian, nv — Navajo, no — Norwegian, nob — Norwegian (Bokmål), nmo — Norwegian (Nynorsk), nov — Novial, oj — Ojibwe, ang — Old English, prg — Old Prussian, os — Ossetian, osd — Ossetian (Digor dialect), osi — Ossetian (Iron dialect), fa — Persian, pl — Polish, pt — Portuguese, ro — Romanian, ru — Russian, sco — Scots, si — Sinhalese, sk — Slovak, sl — Slovene (Slovenian), sr — Serbian, es — Spanish, sv — Swedish, tl — Tagalog, tg — Tajik, ta — Tamil, tt — Tatar, te — Telugu, th — Thai, tpi — Tok Pisin, tr — Turkish, tk — Turkmen, uk — Ukrainian, ur — Urdu, uz — Uzbek, vi — Vietnamese, vo — Volapük, wa — Walloon, cy — Welsh, fy — West Frisian, yi — Yiddish, esu — Yup'ik

<input class="span2" size="16" type="text" value="30-11-2012">

</div>

Methods

.datepicker(options)
Initializes an datepicker.

.datepicker('show')
Show the datepicker.

.datepicker('hide')
Hide the datepicker.

.datepicker('place')
Updates the date picker's position relative to the element.

.datepicker('setValue', value)
Set a new value for the datepicker. It cand be a string in the specified format or a Date object.

Events

Datepicker class exposes a few events for manipulating the dates.

Event Description
show This event fires immediately when the date picker is displayed.
hide This event is fired immediately when the date picker is hidden.
changeDate This event is fired when the date is changed. 
$('#dp5').datepicker()
	.on('changeDate', function(event){
		if (event.date.valueOf() < startDate.valueOf()){
			....
		}
	});
